## Project setup

```bash
$ bun install

# generate nestjs typescript proto definations
$ bun run generate:proto
```

## Run the project

```bash
# watch mode
$ bun run start:dev

# debug mode
$ bun run start:debug
```

## Compile the project

```bash
$ bun run build
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
